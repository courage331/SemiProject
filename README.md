## 개요
* 회원제도를 통한 반려견 등록, 호텔 체크인, 상품 구매, 게시글 작성, 관리자는 상품과 게시글의 추가삭제를 관리할 수 있습니다.
## 공헌한 내용
* 참여도: 25%
* 검색 결과 데이터 출력: JSON
* 웹 화면 구성: HTML, CSS, JavaScript
* 결과를 얻어오기 위한 서버 액션: JAVA, Servlet, JSP
* DB액션 처리: JDBC
* 개발 Tool: Eclipse, Apache Tomcat v9.0, CKediter, kakao
